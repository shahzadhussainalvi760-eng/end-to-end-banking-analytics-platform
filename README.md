# End-to-End Banking Analytics Platform

## Overview
This project is a complete banking analytics solution built using Excel, Power BI, MySQL, and Python. It focuses on transforming raw banking data into meaningful business insights through data cleaning, SQL analysis, KPI reporting, automation, and interactive dashboards.

The project demonstrates real-world data analytics workflow including ETL, database management, exploratory analysis, and business intelligence reporting.
